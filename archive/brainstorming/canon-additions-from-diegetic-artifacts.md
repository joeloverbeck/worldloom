# Canon ADditions From Diegetic Artifacts - New Skill Proposal

We have the following canon-related skills that work perfectly:

- .claude/skills/canon-addition
- .claude/skills/continuity-audit
- .claude/skills/propose-new-canon-facts
- .claude/skills/create-base-world

However, I believe we may have a gap that even propose-new-canon-facts doesn't account for: diegetic artifacts in worlds/[WORLD_NAME]/diegetic-artifacts/ regularly come up with facts and/or new aspects of the world that although necessarily come from the canon and align with it (the diegetic artifact wouldn't have been produced as-is otherwise), the world may benefit from those new facts and/or aspects included in the diegetic artifact to be properly canonized. The proposal, given that incorporating this into propose-new-canon-facts would likely dilute the intention of that skill, would be to create a dedicated skill that allows the user to indicate a world in worlds/[WORLD_NAME], as well as the route to a specific diegetic artifact, and the skill proposes new canon facts that align with how propose-new-canon-facts produces these proposals.
